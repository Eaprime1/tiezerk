# Void Marrow

*The connective tissue of the universe — wormholes and void tubes threading all things.*

---

## Identity

Void Marrow is not a place in the conventional sense. It is the system of voids, channels, and connective structures that runs beneath and through all other environments — the universe's own circulatory system. Wormholes are its valves; void tubes are its veins. To enter Void Marrow is to enter the space between spaces, the medium that allows distant things to be adjacent.

Void Marrow is traversed, not inhabited. It is a passage, not a destination.
